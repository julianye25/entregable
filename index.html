Diferentes formas de incluir css en una pagina web

  1. Estilos internos
  2. Estilos externos
  3. Estilos inline

1.Estilos internos

Forma en la cual require que se incruste la etiqueta style dentro de la etiqueta head

2.Estilos externos

En esta forma se debe de enlazar un archivo css externo para escribir todo el codigo de css en su interior, forma mas eficiente de trabajar con css

3.Estilos inline

Forma en la cual se utiliza para agregar estilos a un elemento html especifico
